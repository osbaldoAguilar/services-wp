Adds Font Awesome 5 (SVG+JS version) to your WordPress website (ver 5.12.0) - 7,702 icons

== Description ==

Once installed, this plugin will allow you to use Font Awesome 5 throughout your website. To find an icon to use, visit [fontawesome.com/icons](https://fontawesome.com/icons?d=gallery).
On your website, just copy/paste the icon code in this format: `<i class="fas fa-mug-hot"></i>`. There is more documentation available here: [fontawesome.com/how-to-use/on-the-web/referencing-icons/basic-use](https://fontawesome.com/how-to-use/on-the-web/referencing-icons/basic-use)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. That's all, now you're ready to start using Font Awesome 5!
